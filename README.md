# Loan Approval & Risk Analysis Dashboard

## 📊 Project Overview
This project analyzes loan application data to identify key factors influencing loan approval decisions using SQL and Power BI.

## 🛠 Tools Used
- PostgreSQL (Data Cleaning & Analysis)
- Power BI (Dashboard & Visualization)

## 📁 Dataset
Loan Prediction Dataset (Kaggle)

## 🔧 Key Steps
- Imported and cleaned data in PostgreSQL
- Handled missing values (credit history, gender)
- Created derived column (credit_flag)
- Performed SQL analysis
- Built interactive Power BI dashboard

## 📊 Dashboard Features
- Total Applications, Approval Rate, Avg Loan Amount
- Approval rate by Credit History
- Loan distribution analysis
- Property area insights
- Interactive filters (Gender, Education, Area)

## 🔍 Key Insights
- Applicants with good credit history have the highest approval rate
- Semi-urban areas show higher loan approvals
- Missing credit history still shows a high approval trend
